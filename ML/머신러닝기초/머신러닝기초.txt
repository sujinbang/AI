머신러닝의 종류
- 분류 
	범주형 데이터 : 종속변수 사용
	연속형 데이터 : 독립변수 사용
- 회귀
	연속형 데이터 : 독립, 종속변수 사용
- 군집
	범주형 데이터 : 독립변수 사용
	연속형 데이터 : 독립변수 사용

지도학습 : 종속변수가 있을 경우
비지도학습 : 종속변수가 없는 경우

군집분석은 비지도학습
선형 회귀분석에는 독립변수가 하나만 들어감

분석방법

1. 데이터 수집
2. 전처리
3. 학습모델 생성
4. 예측(테스트)
5. 최종모델 검정(평가)

- 분류분석 : 의사결정나무(지시적 트리), *랜덤 포레스트, SVM, KMM, *로지스틱 회귀모델
내가 알고 있는 개념을 가지고 모든 모델을 전부 테스트한 다음에 
결정계수(예측을 할 수 있는 정확도, r-square)가 가장 높은 것을 선택

- 회귀분석 : 선형회귀분석, 단순선형회귀분석(잘안씀), *다중선형회귀분석*, *다항회귀분석, 로지스틱 회귀모델

- 군집분석 : Kmean 군집분석, *클러스터링 분석, DB스캔분석

종속변수(정답)가 기준
분류하는 대상이 여러개일때 군집분석을 사용
결정계수에 정해진 기준은 없으나 사안에 따라서 다름 책에선 보통 0.3
